# VUEX-BOILERPLATE 
This demo is created for reduce time developers who want to use vuex to control state



## Installation

```sh
# install dependencies
npm install # or yarn

# on local
npm run local

# run test e2e 
npm run test:e2e 

# build for production
npm run build

```
